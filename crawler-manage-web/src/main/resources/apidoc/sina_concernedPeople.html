查询关注人

说明

查询关注人接口是通过用户微博关注的人展示页面来查询关注人概要信息的接口。

URL

http://<Project>/api/weibo/follower?url=

格式

JSON

HTTP请求方式

GET

接口请求参数

# 必选 类型及范围 说明
url true string 用户用户关注的人列表页面（最多5页）
isDebug false boolean 是否启用调试模式

注意事项

无

接口例子

说明：非调试模式下，查询id为“1656627870”的微博用户的第二页的关注者。默认每页20人。

请求：http://<Project>/api/weibo/follower?url=http://weibo.com/1656627870/follow?page=2

返回：JSON
