搜索用户

说明

用户查询接口通过知乎昵称来查询知乎相关用户的基本信息的接口。

URL

http://<Project>/api/zhihu/user?q=

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
q true string 查询条件(需要对查询条件进行一次utf8编码)
isDebug false boolean 是否启用调试模式

注意事项

无

例子

说明：非调试模式下，根据用户知乎姓名来查询相应用户。

请求：http://<Project>/api/zhihu/user?q=%e8%b0%a2%e7%86%8a%e7%8c%ab%e5%90%9b

返回：JSON

返回字段说明

返回正确data结果的字段说明对照表：
返回值字段 字段类型 字段说明
name string 用户的昵称
bio string 一句话介绍
profile_image string 用户头像
gender string 用户性别
profile string 用户主页连接
ask_num string 用户的提问次数
answers_num string 用户的回答问题总数
posts_num string 用户的发表文章总数
followers_num string 对该用户的关注人数
