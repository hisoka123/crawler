查找用户信息

说明

查找用户信息接口是通过被查询领英用户的主页链接来查询该用户的详细信息的接口。

URL

http://<Project>/api/linkedin/userinfo?url=

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
url true string 被查询用户的查询主页链接（profile）
isDebug false boolean 是否启用调试模式

注意事项


1. url参数可以通过用户查询接口得到（profile字段），但需要用utf8字符集对profile进行编码
2. 由于Linkedin平台本身信息的丰富性以及用户注册信息的特点，每个用户的信息可能呈现出各自不同的特点（比如用户A中拥有的字段在用户B中不一定能够找到），但每个用户信息的数据结构基本上是一致的。
