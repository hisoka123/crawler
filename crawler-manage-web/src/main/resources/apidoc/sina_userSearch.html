搜索用户

说明

用户搜索接口是通过微博昵称来查询微博用户的接口。

URL

http://<Project>/api/user?q=

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
q true string 查询条件(需要对查询条件进行一次utf8编码)
isDebug false boolean 是否启用调试模式

注意事项

无

例子

说明：非调试模式下，查询微博昵称为“唐沐thomas”的用户

请求：http://<Project>/api/weibo/user?q=%E5%94%90%E6%B2%90thomas

返回：JSON

返回字段说明

返回正确data结果的字段说明对照表：
返回值字段 字段类型 字段说明
uid string 用户的ID
nickname string 用户的昵称
follow string 用户关注的人数
follow_url string 用户关注的人的列表页面
fans string 粉丝数量
fans_url string 粉丝列表页面
statuses string 用户发帖总数
statuses_url string 用户发帖的列表页
profile string 用户首页
person_info string 用户简介
profile_image string 用户头像图片链接
location string 用户所在地
gender string 性别
person_label string[] 用户标签
person_job string[] 用户的工作经历标签
person_edu string[] 用户的教育经历标签
